You can display commands as well:

```
$ birb create new-bird
```

Writing equations is supported as long as you know Latex syntax. You can use it in inline mode: $c = \sqrt{a^2 + b^2}$ or as a block:

$$
\theta = \begin{pmatrix}
   a & b \\
   c & d
\end{pmatrix}
$$
